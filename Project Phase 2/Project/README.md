# Rule Lawyers

Rule Lawyers is a simple fictional board-game shop website created for the PHP Phase One assignment. 
The project includes the required pages, shared templates, and basic styling.

## Features in Phase One
- Four public pages: Home, Shop, About, Contact
- Header and footer included on every page 
- A consistent layout and basic custom CSS
- Organized folder structure with separate directories for public pages, UI templates, styles, and database placeholders

## Folder Structure
project/
├─ public/          # Main PHP pages
├─ ui/              # Header and footer templates
├─ styles/          # CSS files
├─ database/        # Placeholder for future database work
└─ README.md
